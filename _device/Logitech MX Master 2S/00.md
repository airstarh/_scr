## 🎯 Что именно решило проблему?

**Одна единственная команда:**
```bash
solaar config "Wireless Mouse MX Master 2S" gesture2-divert 46 False
```

Она отключила "перехват" жеста 46 (бокового колесика) в Solaar, позволив событиям идти напрямую в систему.

## 📋 Что нам НЕ помогло (и почему):

| Инструмент | Результат | Причина |
|------------|-----------|---------|
| **libinput quirks** | ❌ Бесполезно | Проблема была на уровне драйвера, не libinput |
| **logiops** | ❌ Не сработал | Не поддерживает Gesture 2 на MX Master 2S |
| **xinput** | ❌ Не применим | Вы работаете на Wayland, xinput для X11 |

## ✅ Что реально работает и нужно:

**Только Solaar** (и то лишь одна настройка).

Вот минимальный набор для полной работы мыши:

```bash
# 1. Установить Solaar
sudo apt install solaar

# 2. Отключить перехват жеста 46
solaar config "Wireless Mouse MX Master 2S" gesture2-divert 46 False

# 3. (Опционально) Инвертировать направление прокрутки, если нужно
solaar config "Wireless Mouse MX Master 2S" hires-smooth-invert True
```

**Больше ничего не требуется!**

## 🗑️ Что можно удалить (мешает или не нужно):

```bash
# Удалить logiops (если установлен)
sudo apt remove logiops

# Удалить старый проблемный файл правил
rm -f ~/.config/solaar/rules.yaml

# Удалить libinput quirks файл (не сработал)
sudo rm -f /etc/libinput/local-overrides.quirks
```

## 📦 Как управлять при переустановке ОС:

Вот **чистая инструкция** для нового компьютера с Kubuntu/KDE Plasma на Wayland:

### После чистой установки:

```bash
# 1. Установить Solaar
sudo apt update
sudo apt install solaar

# 2. Подключить мышь по Bluetooth

# 3. Выполнить настройку
solaar config "Wireless Mouse MX Master 2S" gesture2-divert 46 False

# 4. (Опционально) Инвертировать направление
solaar config "Wireless Mouse MX Master 2S" hires-smooth-invert True

# 5. Готово! Больше ничего не нужно
```

### Проверка, что всё работает:

```bash
solaar config "Wireless Mouse MX Master 2S" | grep gesture2
# Должно показать: gesture2-divert = {46: False}
```

## 🤔 Так Solaar полезен или нет?

**ДА, но только для одной конкретной цели:**

- ✅ **Полезен:** Отключить перехват жеста 46 (это решило проблему)
- ✅ **Полезен:** Инвертировать направление прокрутки
- ✅ **Полезен:** Смотреть уровень заряда батареи
- ❌ **НЕ полезен:** Создание правил (rules.yaml) для переназначения — на Wayland это глючно

**Solaar вам нужен.** Без него боковое колесико не работает.
