## SCANNER

Для сканирования с МФУ Pantum M6500 в Kubuntu 25 с KDE Plasma 6 потребуется установить необходимые драйверы и настроить систему сканирования. Основные шаги включают установку драйверов, настройку Sane и проверку подключения.

## Установка драйверов

1. **Скачайте драйвер с официального сайта Pantum.** На сайте global.pantum.com доступен драйвер для серии M6500, включая модель M6500NW PLUS. Версия драйвера — V1.1.167, размер файла — 8,32 МБ. [```7```](https://global.pantum.com/search/download-driver/?s=Pantum%20M6500)

2. **Распакуйте архив с драйвером.** Если драйвер скачан в формате `.tar.gz`, используйте команду:
   ```bash
   tar xvzf Pantum_Driver_V1.1.167.tar.gz
   ```

3. **Установите драйвер.** Перейдите в распакованную папку и запустите установочный скрипт:
   ```bash
   cd Pantum_Driver_V1.1.167
   ./install.sh
   ```

## Установка компонентов для сканирования

Для работы сканирования потребуется установить пакеты `sane` и `sane-utils`:
```bash
sudo apt install sane sane-utils
```

Также может потребоваться библиотека `libjpeg8`. Если она отсутствует, установите её вручную, скачав пакет `libjpeg8_8b-1_amd64.deb` и выполнив:
```bash
sudo dpkg -i libjpeg8_8b-1_amd64.deb
```

## Настройка Sane

1. **Добавьте поддержку Pantum в конфигурацию Sane.** В файл `/etc/sane.d/dll.conf` добавьте строку:
   ```
   pantum6500
   ```

2. **Определите сетевой адрес принтера.** Если принтер подключён по сети, используйте утилиту `avahi-browse-domains -alr`, чтобы найти его доменный адрес (например, `Pantum-XXXXXX.local`). Добавьте этот адрес в файл `/etc/sane.d/net.conf`.

3. **Запустите службу saned:**
   ```bash
   sudo systemctl start saned.socket
   ```

## Проверка доступности сканера

Чтобы проверить, распознаётся ли сканер, выполните команду:
```bash
scanimage -L
```
В выводе должна появиться строка, указывающая на доступность сканера Pantum.

## Использование сканера

Для сканирования можно использовать различные программы, например:
- **xsane** — графический интерфейс для Sane;
- **gscan2pdf** — программа для сканирования и сохранения в PDF;
- **NAPS2** — программа для сканирования документов и изображений.

Если xsane не установлен, установите его командой:
```bash
sudo apt install xsane
```

## Решение распространённых проблем

| Проблема | Возможное решение |
|---|---|
| Сканер не распознаётся | Проверьте, что принтер включён и подключён к сети. Убедитесь, что в `/etc/sane.d/net.conf` указан правильный адрес принтера. |
| Ошибки при запуске scanimage | Проверьте права доступа к устройству. Возможно, потребуется добавить пользователя в группу `lp` или `scanner`. |
| Нет изображения при сканировании | Проверьте настройки сканирования в программе (разрешение, режим и т. д.). Убедитесь, что в лотке есть бумага (если требуется). |

Если проблема не решается, проверьте логи системы (например, через `journalctl`) на наличие ошибок, связанных с Sane или CUPS.

Обратите внимание, что инструкции могут незначительно отличаться в зависимости от конкретной версии системы и настроек сети. Если возникнут сложности, уточните детали — помогу разобраться!
